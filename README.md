# Overview

Soundproof is a modern-aim at smart cities, focused on improving citizens' lives through taking care of their hearing. This is done by the two main functionalites this app provides:

- Heatmap, with city spaces marked in colour to represent the noise levels
- Audio recording, through the in-built smartphone microphones allowing users to contribute to the quality of the app.
